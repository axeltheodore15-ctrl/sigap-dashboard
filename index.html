📝 Executive Summary
HARI INI

Pada 05 Maret 2026, pemantauan media sosial dan media online terkait Polri menunjukkan total 46.390 mention dalam 24 jam terakhir, meningkat 18% dibanding hari sebelumnya. Isu paling mendominasi adalah penembakan warga Bertrand oleh oknum IPTU di Makassar dengan sentimen negatif 81%, diikuti persiapan Operasi Ketupat 2026 dengan respons positif 72%, serta demo Aliansi Reformasi Polri (AROMI) di Mabes Polri yang memicu viral video mahasiswa membentak polisi. Sentimen keseluruhan mencapai 35% positif, 27% netral, dan 38% negatif. Polri perlu segera menyikapi kasus Makassar dengan langkah transparan dan memperkuat narasi positif Operasi Ketupat menjelang mudik Lebaran.

🚀 Cara Update Dashboard Harian di Netlify
1
Buka dashboard ini setiap pagi, klik "Pindai Sekarang" untuk mendapatkan data terbaru dari AI. Hasil scan otomatis tersimpan di memori sesi.
2
Klik tombol "Unduh Laporan PDF" di bagian bawah halaman untuk mengunduh laporan harian lengkap. File siap dijadikan bahan rapat/briefing.
3
Deploy ke Netlify: Upload file index.html ini ke repository GitHub Anda, lalu hubungkan ke Netlify. Dashboard akan selalu tersedia di URL publik Anda.
4
Untuk update otomatis harian: Aktifkan Netlify Scheduled Functions atau gunakan GitHub Actions dengan schedule cron: '0 1 * * *' (setiap pukul 08:00 WIB) untuk trigger re-deploy otomatis.
5
Tambahkan variabel environment ANTHROPIC_API_KEY di Netlify Dashboard → Site Settings → Environment Variables untuk mengaktifkan fitur AI Scan otomatis.
